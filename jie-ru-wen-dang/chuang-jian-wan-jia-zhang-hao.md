# 创建玩家帐号

## 1) 请求地址

> URL: {APIURL}/api/v1/player/create

## 2）请求参数:

| 参数名    | 类型            | 描述                                        |
| ------ | ------------- | ----------------------------------------- |
| UserID | string\[4-40] | 运营商的玩家唯一标识（注意不能填特殊字符，例如"。建议仅使用数字，英文，下划线。） |

* 示例：

```json
{
  "UserID": "abc"
}
```

## 3) 返回结果:

| 参数名 | 类型    | 描述       |
| --- | ----- | -------- |
| Pid | int64 | 平台玩家唯一标识 |

* 示例：

```json
{
  "error": "",
  "data": {
    "Pid": 100064
   }
}
```

* 重复创建同一个玩家, 返回相同的结果
* /api/v1/player/transferIn 和 /api/v1/game/launch 会自动创建玩家
